# Portafolio
A continuación se presenta un portafolio inspirado en diferentes diseños. Cuenta con cuatro páginas:
- Inicio
- Sobre la persona
- Proyectos
- Contacto

## Tecnologías
- React
- React Router
- Vite
- EmailJS
- Lucide React

## Requisitos    
Tener instalado node.js y pnpm

## Instalación

1. Clone el repositorio
```bash
    git clone https://github.com/Heilong-0/Portafolio.git
```
2. Instale las dependencias
```bash
    pnpm install 
```
3. Levante el servidor
```bash
    pnpm run dev
```
